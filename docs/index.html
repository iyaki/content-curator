Sobre esta página

Hace algún tiempo compre en NIC Argentina el dominio iyaki.ar. El primo paso obvio luego de eso era montar una página web (y si sobraba tiempo configurar un servidor de correo electrónico).

Después de múltiples búsquedas en Google sobre servicios de hosting y demás terminé creando esta página utilizando:

  • Github Pages para el alojamiento web.
  • Cloudflare para el registro de los DNS.
  • Mis manos y el Visual Studio Code (con algunas extensiones) para escribir el código fuente de esta página
Eso es todo. No hay paso de compilación o transpilación, no hay frameworks, ni siquiera librerías de CSS o JS.

En esta primer versión de la página eso puede parecer una obviedad, pero mi intención es seguir desarrollandola de manera incremental ya agregando paulatinamente, estilos, funcionalidad y nuevas secciones.

Por el momento me centro únicamente en que el uso de los elementos HTML sea semánticamente adecuado.

Mi objetivo al crear esta página no es contar con un portfolio para agregar en mi CV sino poder compartir mis experiencias y vivencias en el campo sistemas de información.

Si algo de lo que ven les interesam tienen sugerencias, criticas o cualquier comentario pueden escribirme a: (finalmente no configure un servidor de correo electrónico pero si redirección).
