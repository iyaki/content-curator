Colección personal de artículos de IT y afines

Aquí recopilo artículos que me resultan útiles o interesantes.

Este proyecto fue desarrollado con:

  • Visual Studio Code y mis manos para todo el código HTML y CSS.
  • Notion como CMS.
  • ChatGPT (mediante la API REST de OpenAI) para escribir los comentarios de los artículos.
  • Node.js y algunas bibliotecas para agregar a la web los artículos registrados en el CMS (Código fuente).
  • Github Actions para automatizar el agregado de los artículos.

Patterns for Managing Source Code Branches

El código fuente es un activo vital para cualquier equipo de desarrollo de software, y a lo largo de los años se han desarrollado herramientas de gestión de código fuente para mantenerlo en buen estado. Estas herramientas permiten rastrear los cambios, de manera que se puedan recrear versiones anteriores del software y ver cómo se va desarrollando con el tiempo. También son esenciales para coordinar un equipo de varios programadores que trabajan en un código común. Al registrar los cambios que cada desarrollador realiza, estos sistemas pueden hacer un seguimiento de múltiples líneas de trabajo y ayudar a fusionarlas.
