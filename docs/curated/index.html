Colección personal de artículos de IT y afines

Aquí recopilo artículos que me resultan útiles o interesantes.

Este proyecto fue desarrollado con:

  • Visual Studio Code y mis manos para todo el código HTML y CSS.
  • Notion como CMS.
  • ChatGPT (mediante la API REST de OpenAI) para escribir los comentarios de los artículos.
  • Node.js y algunas bibliotecas para agregar a la web los artículos registrados en el CMS (Código fuente).
  • Github Actions para automatizar el agregado de los artículos.

More product, fewer PMs

Cuántos Product Managers (PMs) son necesarios en un equipo de producto ha sido motivo de debate. Después de que Airbnb reestructuró su organización de gestión de productos, seguido de artículos sobre cómo las startups prosperan sin PMs, la conversación ganó impulso. Estoy de acuerdo en que muchas empresas contratan demasiados PMs, lo que lleva a que se ocupen principalmente de tareas de gestión de proyectos y actualización de planificaciones en lugar de optimizar el producto para el negocio. La necesidad de un PM depende del producto, el equipo y las necesidades de la empresa, por lo que la noción tradicional de una relación 1 a 1 entre equipos y PMs debe reconsiderarse.
