--- title: "Patterns for Managing Source Code Branches" comment: "El código fuente es un activo vital para cualquier equipo de desarrollo de software, y a lo largo de los años se han desarrollado herramientas de gestión de código fuente para mantenerlo en buen estado. Estas herramientas permiten rastrear los cambios, de manera que se puedan recrear versiones anteriores del software y ver cómo se va desarrollando con el tiempo. También son esenciales para coordinar un equipo de varios programadores que trabajan en un código común. Al registrar los cambios que cada desarrollador realiza, estos sistemas pueden hacer un seguimiento de múltiples líneas de trabajo y ayudar a fusionarlas." link: "https://martinfowler.com/articles/branching-patterns.html" tags: curatedContent ---
